# GistEdit

Simplest gists editor with live markdown preview.

## Install

1. `npm i && npm run build`
2. `npm start` or `PORT=3000 npm start`
3. open `localhost:3000/id=<GIST_ID>&name=<GIST_FILE_NAME>` in your browser
